```shell
Ctrl + Shift + A ：查找动作，可以用来查找快捷键

Ctrl + N : 寻找类文件

Ctrl + Shift + N ：在项目中寻找文件

Ctrl+Shift+方向键：上移/下移代码一行

Ctrl + 空格：自动补全

Alt + F7 ： 在整个项目中寻找类名、方法名以及变量名

Ctrl + Q ： 快速查看类或方法名的文档

Ctrl + B （Ctrl + 鼠标左键）： 定位到类、方法或变量的声明

Ctrl + F12：快速定位到被修改的文件

Shift + F6 ：重命名类名、方法名以及变量名

Ctrl + O ： 重写父类方法

Ctrl + I  ：实现父类接口

Ctrl + Shift + 空格 ：查找合适的方法或变量

Ctrl + Alt + L ：自动格式化代码

Ctrl + X ：删除一行

Alt + insert ： generate

Ctrl + Shift + F ：全局搜索

Ctrl + Shift + R ：全局替换

Ctrl + Alt + 方向键：返回上次浏览位置

Ctrl + Shift + U ：切换大小写

Ctrl + Shift + - ：收缩所有代码

Ctrl + Shift + = ：2

Ctrl + Alt + t ：surround with

Ctrl + / : 行注释

Ctrl + shift + / : 块注释

Ctrl + Alt + M : 提取当前选择为方法

Ctrl + Alt + C : 提取当前选择为常量

Ctrl + Alt + V : 提取当前选择为变量

```
